# 网络基础
## ip
>公网ip-以太网-百度ip
局域网ip-本地网络地址-ipconfig
本机ip-所有机器都一样-127.0.0.1或localhost
## 域名
>www.?????.com类型网址对应于ip,便于记忆的结果
## 端口
>端口号:默认为80,既等同于未输入表现形式为ip+:+端口(127.0.0.1:8080 tomcat的本机端口)
8080 tomcat默认端口
    3306 mysql默认端口
一个电脑有65535个端口
# 网络协议
## tcp
>- 发送消息相对更慢
>- 发送消息的过程非常可靠,不会发错
>- 三次握手(发送消息前的确认流程)
## udp
>- 发送速度相对超快
>- 直播用的udp协议
>- 发送数据不可靠,容易出错
## http
>- 基于tcp制作
>- http
1.发送消息未加密,任何人都能看到
2.不安全
>- https
1.通过工信部颁布的证书,实现了加密的过程
2.安全
